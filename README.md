# playlists

' Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
' Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan omia äänitiedostoja.
' Käyttäjä pystyy lisäämään äänitiedostoja soittolistoihin.
' Käyttäjä näkee sovellukseen lisätyt äänitiedostot ja soittolistat.
' Käyttäjä pystyy etsimään äänitiedostoja ja soittolistoja hakusanalla.
' Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja ja käyttäjän lisäämät äänitiedostot.
' Käyttäjä pystyy valitsemaan äänitiedostolle yhden tai useamman luokittelun (esim. musiikin genre, ääniefekti, keskustelu).
' Käyttäjä pystyy arvostelemaan ja kommentoimaan sovellukseen lisättyjä äänitiedostoja. Kommentissa näkyy käyttäjätunnus.
